Admin Guide — Linking Documents

To make a document title clickable on the Files & Docs page, open the Documents sheet and fill in one of these columns for that row:

  • FileURL — paste the file's Google Drive shareable link directly (Drive → Share → Copy link). This takes priority if both columns are filled.
  • DriveFileID — or just the Drive file ID (the long string in the file's URL between /d/ and /view) if you don't want to paste the full link.

If neither column is filled in, the title shows as disabled with a "No link available" tooltip instead of a broken link.

Sharing permissions — set the Drive file's sharing to match the AccessLevel column:

  • AccessLevel = All → share as "Anyone with the link can view."
  • AccessLevel = Board → restrict sharing to the gorock.org Google Workspace domain. These show a 🔒 next to the title in the app.
